# Ultimate Alex

#### Product Description:

> Fully responsive online shop based on vue stack.

## Demo

Please click here for a live demo.

## Tool Chain:

- Base: Vue
- UI: Bootstrap
- State Management: Vuex
- Routing: Vue-Router
- Form Validation: VeeValidate
- API: vuex-rest-api

## Features:

- [x] Login/Signup
- [x] ShoppingCart
- [x] Product Filtering(filtered by category, price, tags)
- [x] Advanced Product Search

## Get started

#### Project setup

```
npm install
```

#### Run project in dev mode and start mock backend service

```
npm run back
npm run serve
```

#### Compiles and minifies for production

```
npm run build
```

#### Lints & fixes files

```
npm run lint
```

#### Recompile SVG icons

```
npm run svg
```
