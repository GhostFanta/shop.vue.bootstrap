## Get started
#### Project setup
```
npm install
```
#### Run project in dev mode and start mock backend service
```
npm run back
npm run serve
```
#### Compiles and minifies for production
```
npm run build
```
#### Lints & fixes files
```
npm run lint
```

#### Recompile SVG icons
```
npm run svg
```
